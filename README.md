# Assistant-Personnel-pour-la-Gestion-du-Temps

Aider l’utilisateur à organiser automatiquement son emploi du temps en fonction de ses habitudes, de ses priorités et de son historique d’activité.

## Import du dépôt demandé

Si vous voulez récupérer le dépôt suivant :

- `https://github.com/yassinebenacha/multilingual-noise-demo.git`

utilisez le script ci-dessous :

```bash
bash scripts/import_multilingual_noise_demo.sh
```

Vous pouvez aussi préciser un dossier cible :

```bash
bash scripts/import_multilingual_noise_demo.sh ./vendor/multilingual-noise-demo
```
